# NEXT_STEPS_GUIDE — .md to .tex arXiv/PRX Conversion

This guide covers the conversion of VIM manuscript Markdown files to LaTeX for arXiv and Physical Review X (PRX) submission.

---

## 1. Pandoc Conversion

### Basic command

```bash
pandoc section.md -o section.tex --wrap=preserve
```

- `--wrap=preserve` keeps long lines (equations, tables) intact.
- Use `-t latex` for explicit LaTeX output.

### Math extensions

Ensure `\[ \]` and `\( \)` survive conversion:

```bash
pandoc section.md -o section.tex -t latex --mathjax
```

Or with native math:

```bash
pandoc section.md -o section.tex -f markdown+tex_math_dollars -t latex
```

### Batch conversion

```bash
for f in vim_section_*.md; do pandoc "$f" -o "${f%.md}.tex" --wrap=preserve; done
for f in appendix_*.md; do pandoc "$f" -o "${f%.md}.tex" --wrap=preserve; done
```

## 2. arXiv/PRX LaTeX Template

### Document class

```latex
\documentclass[11pt,oneside]{article}
% For PRX: \documentclass[prx,reprint]{revtex4-2}
```

### Required packages

```latex
\usepackage[margin=1in]{geometry}
\usepackage{amsmath,amssymb,amsthm}
\usepackage{graphicx}
\usepackage{hyperref}
\usepackage{lmodern}
\usepackage[T1]{fontenc}
\usepackage[utf8]{inputenc}
```

### Hyperref setup

```latex
\hypersetup{
  colorlinks=true,
  linkcolor=cyan,
  urlcolor=magenta,
  citecolor=orange,
  pdftitle={VIM Manuscript},
  pdfauthor={Ross A. Edwards}
}
```

---

## 3. Equation Preservation

- **Display math:** Use `\[ ... \]` or `\begin{equation}...\end{equation}`.
- **Inline math:** Use `\( ... \)` or `$...$`.
- Pandoc with `-f markdown+tex_math_dollars` preserves `$...$` and `$$...$$`.
- After conversion, verify that `\beta`, `\Phi_{\text{HIF}}`, `\nabla`, etc. render correctly.

## 4. Figure Inclusion

### Paths

Figures are generated by `fig_scripts/fig_XXX_*.py` and saved as PNG. Paths should be relative to `tex_output/`:

```latex
\includegraphics[width=0.8\textwidth]{fig_001_001.png}
```

### Figure directory layout

Place generated figures in `tex_output/` or a subdirectory:

```
tex_output/
  fig_001_001.png
  fig_002_002.png
  ...
  fig_083_083.png
```

### Generating all figures

```bash
cd main/vim/fig_scripts
for f in fig_*.py; do python "$f"; done
```

Or use a batch script to run all and copy outputs to `tex_output/`.

## 5. Bibliography

### BibTeX style

Common choices: `plain`, `unsrt`, `alpha`, `apalike`.

```latex
\bibliographystyle{plain}
\bibliography{vim_refs}
```

### .bib file structure

```bibtex
@article{edwards2026vim,
  author  = {Edwards, Ross A.},
  title   = {Vacuum Impedance Matching and the Theory of Balance},
  journal = {arXiv preprint},
  year    = {2026},
  eid     = {arXiv:xxxx.xxxxx}
}
```

### natbib (optional)

```latex
\usepackage{natbib}
\citep{edwards2026vim}
\citet{edwards2026vim}
```

## 7. Style Checklist

- [ ] **Fonts:** `lmodern` (or `times` for PRX)
- [ ] **Margins:** 1 inch (`geometry`)
- [ ] **Hyperref colors:** Links, URLs, citations
- [ ] **Equation numbering:** `\numberwithin{equation}{section}` if desired
- [ ] **Figure captions:** `\caption{...}` and `\label{fig:...}` for cross-references
- [ ] **Consistent notation:** β, Φ_HIF, x, etc. per EQUATION_EXTRACTION_SUPPLEMENT
