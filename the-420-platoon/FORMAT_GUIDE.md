# 📐 The 420 Platoon — Format Guide

---

## Visual Identity

**Primary Palette:**
- Background: Deep military olive / void black hybrid
- Accent 1: Haze Gold (#C9A84C) — smoke, warmth, nostalgia
- Accent 2: Platoon Cyan (#00E5FF) — tech, dystopia, Aurphyx DNA
- Accent 3: Claymation Red (#FF1744) — danger, Dr. Satan, combat
- Text: Crystal White (#E8F0FF)

**Typography:**
- Title Card: Bebas Neue Bold — ALL CAPS, military stencil energy
- Pop-Up Video bubbles: Rounded 90s sans-serif (Comic Sans adjacent but NOT Comic Sans)
- Dr. Satan exposition: Distressed serif, slight chromatic aberration
- Historical claymation name cards: Slab serif, like a wrestling entrance card

---

## The Pop-Up Video System

**Trigger:** Random intervals throughout main action (never during emotional beats)
**Sound:** Iconic "WHOOP WHOOP" bubble pop from VH1 original
**Content types:**
- Obscure historical trivia relevant to scene
- Meta commentary on the show itself
- Absurd statistics
- Lore drops
- Punchline extensions

**Rule:** Pop-Ups NEVER explain the joke. They ADD to it or misdirect from it.

---

## Rendering Specifications

| Element | Style | Notes |
|---|---|---|
| Main cast | Hyper-realistic 3D | Uncanny valley is intentional |
| Claymation characters | Physical clay aesthetic | Celebrity Deathmatch reference |
| Pop-Up bubbles | 2D flat overlay | 90s VH1 exact style |
| Vinyl deck transitions | Live action or 2D composite | Always 8-12 seconds |
| Dr. Satan segments | Claymation only | Never realistic |
| Historical Break 2 | Mixed — avatar + clay | Two different worlds in frame |
